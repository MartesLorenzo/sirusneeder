Política de Privacidade Sirus Needer

Última atualização: Março 2026

Este aplicativo foi desenvolvido para ajudar usuários a realizar cálculos de gastos e ganhos do dia a dia, como compras de supermercado e controle financeiro simples.

1. Coleta de Informações

Este aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários.

Nenhuma informação como nome, email, localização ou dados do dispositivo é recolhida.

2. Uso do Aplicativo

Todas as funcionalidades do aplicativo são executadas localmente no dispositivo do usuário, sem necessidade de conexão com servidores externos.

3. Compartilhamento de Dados

Como não há coleta de dados, nenhuma informação é compartilhada com terceiros.

4. Segurança

O aplicativo é projetado para funcionar de forma segura, garantindo que nenhuma informação do usuário seja armazenada ou transmitida.

5. Direitos do Usuário

Como não há coleta de dados pessoais, não há armazenamento de informações que possam ser solicitadas, alteradas ou removidas.

6. Alterações nesta Política

Esta política pode ser atualizada futuramente. Recomendamos que o usuário revise esta página periodicamente.
